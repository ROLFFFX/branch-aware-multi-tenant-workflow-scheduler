### Frontend - Branch-Aware Multi-Tenant Workflow Scheduler

This frontend is built with React + Vite

### Requirements

Node.js 20+ (Vite 5+ requires Node ^20.19 or newer)

- Check your Node version `node -v`
- If below 20, install via nvm:

```
nvm install 20
nvm use 20
```

### Setup

Install dependencies via

```
npm install
```

Start development server via

```
npm run dev
```
